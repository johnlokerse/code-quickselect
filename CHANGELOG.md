# Changelog

## 1.1.0

- Added Quick Select support for Bicep `param` and `output` declarations
- Clicking Quick Select on a param or output selects all declarations of that kind at once

## 1.0.0

- First stable release
- CodeLens-based block selection for Bicep, PowerShell, C#, and Markdown
- Handles nested braces and skips strings/comments

## 0.0.1

- Initial implementation of CodeLens and block selection
